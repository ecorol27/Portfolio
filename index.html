Cadre :
Projet de fin de première année en algorithmique C

Description :
Le but de ce projet était de réaliser une filmothèque qui puisse analyser une demande d'un utilisateur et renvoyer les résultats obtenues. Jaymerson Bonder


Compétences :
Acting, Travail de groupe, montage

Participants :
3 personnes

Rôles :
Scénariste, Metteur en scène

Date :
2023 - 1ère année d'études

Projet de Formation Humaine et Sociale

Description :
Dans le cadre de nos cours de Formation Humaine & Sociale nous devions réaliser un court métrage sur le thème de l'aventure. Ce court métrage nous plonge dans la vie de Jaymerson, un rebelle qui, avec son groupe, attaque une base gouvernementale abritant une IA contrôlant la société ainsi qu'un dictateur. Leur objectif est de détruire cette IA pour sauver la société. Nous avons réaliser notre projet en stop-motion grâce à des légos pris en photo sur des fonds verts construits avec du cartons. Nous avons ensuite fait du montage vidéo afin d'obtenir le résultat attendu.

Le jeu de la vie


Compétences :
Langage C

Participants :
1 personne

Date :
2023 - 1ère année d'études

Cadre :
Cours d'Algorithmique C

Description :
Nous devions dans le cadre d'un cours en Langage C réaliser un programme nous permettant de visualiser la création de plusieurs entités en fonction des différentes générations du programme. Le code a le même principe que celui du célebre jeu de la vie. C'est un jeu à zéro joueur, ce qui signifie que son évolution est déterminée par son état initial, sans nécessiter d'autres entrées. Le jeu de la vie fonctionne en suivant des règles simples basées sur l'état des cellules voisines : les cellules naissent, survivent ou meurent en fonction de leur environnement.
